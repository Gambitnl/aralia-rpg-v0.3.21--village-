# Weaver’s Tools (1 GP)
**Ability:** Dexterity 
**Weight:** 5 lb.
**Utilize:** Mend a tear in clothing (DC 10), or sew a Tiny design (DC 10)
**Craft:** Padded Armor, Basket, Bedroll, Blanket, Fine Clothes, Net, Robe, Rope, Sack, String, Tent, Traveler's Clothes
