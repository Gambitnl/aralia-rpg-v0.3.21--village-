# Tool Proficiency
If you have proficiency with a tool, add your Proficiency Bonus to any ability check you make that uses the tool. If you have proficiency in a skill thatâ€™s used with that check, you have Advantage on the check too.

Your features might give you proficiency with a tool. A monster has proficiency with any tool in its stat block.
